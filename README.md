# Simple JavaScript Calculator

A clean, responsive, and beginner-friendly web-based calculator built using HTML, CSS, and JavaScript. This project is designed to demonstrate basic DOM manipulation and arithmetic logic in JavaScript.

## Features
- **Basic Arithmetic:** Supports addition, subtraction, multiplication, and division.
- **Clear Functionality:** Reset the display using the `C` button.
- **Delete Function:** Remove the last entered character (Backspace).
- **Responsive Design:** Works on desktops, tablets, and mobile devices.
- **Modern UI:** Clean interface with hover effects and shadow styling.

## Technologies Used
- **HTML5:** For the structure of the calculator.
- **CSS3:** For styling, layout (Flexbox/Grid), and responsiveness.
- **JavaScript (ES6):** For handling calculations and user input logic.

## How to Run
1. Clone this repository or download the source code.
2. Navigate to the project folder.
3. Open `index.html` in any modern web browser.

## Project Structure
- `index.html` - The main structure of the calculator.
- `style.css` - The visual styling and layout.
- `script.js` - The logic for calculations and button interactions.
